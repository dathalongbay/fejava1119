thuộc tính id của thẻ html
        thuộc tính id được sử dụng để định danh cho 1 thẻ html là duy nhất
        không được phép có 2 thẻ có id trùng nhau trên 1 trang web
        hiểu đơn giản id như số chứng minh thư

        thuộc tính id có ý nghĩa quan trọng trong css và javascript

        Trong css sẽ có 1 khái niệm là bộ chọn ( selector )
        Bộ chọn này khi muốn chọn 1 thẻ html dựa vào thuộc tính id của html
        thì sẽ viết là #id
        ví dụ trong file này :
        #target1
        #target2
        #target3
