Thuộc tính position trong css
        Thuộc tính quyết định vị trí của 1 thẻ trong css
        Trước khi tìm hiểu thuộc tính position mình sẽ 4 thuộc tính trong css
        để đặt vị trí cho 1 thẻ :
        trong trình duyệt hệ tọa độ theo pixel hay %
        top ( bên trên ) left ( bên trái ) right ( bên phải )
        bottom ( tọa độ bên dưới )
        ví dụ :
        .classA {
            position: absolute;
            top: 20px;
            left: 30px;
            right: 10px;
            bottom: 20px;
        }
        Khi đặt toạn độ thông thường chỉ cần 1 tọa độ theo chiều dọc và 1 tọa độ
        theo chiều ngang .
        .classA1 {
            position: absolute;
            top: 20px;
            left: 30px;
        }

        .classA2 {
            position: absolute;
            top: 20px;
            right: 30px;
        }

        .classA3 {
            position: absolute;
            bottom: 20px;
            right: 30px;
        }

        .classA4 {
            position: absolute;
            bottom: 20px;
            left: 30px;
        }

        top và bottom đại diện tọa độ theo chiều dọc
        left và right đại diện tọa độ theo chiều ngang
