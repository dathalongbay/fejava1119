Các giá trị của thuộc tính postion trong css
        - static ( tĩnh ) đây là giá trị mặc định position
        - relative ( tương đối )
        - absolute ( tuyệt đối )
        - fixed ( cố định )
        - sticky ( gần giống cố định )
    
        Trong 1 trang web vị trí của 1 thẻ html sẽ phụ thuộc vào 1 số
        thuộc tính quan trọng sau :
        - phụ thuộc vào box model ( margin )
        - phụ thuộc vào thuộc tính display: block hay inline hay inline-block
        - phụ thuộc vào thuộc tính float ( trôi ): left hay right
        cuối cùng phụ thuộc thuộc tính quan trọng nhất là position
        position mặc định của các thẻ là static
        ví dụ :
        p {
            position: static;
            float: left;
            margin: 20px;
            display: block;
        }
