Cách thuộc tính nền trong CSS
        1. background-color ( màu nền )
        2. background-image ( ảnh nền )
        3. background-repeat ( có nên lặp ảnh nền khi mà kích thước lớn
        hơn ảnh nền )
        4. background-position ( xác định vị trí của ảnh nền )
        5. background-attachment ( ảnh nền có cố định trong 1 thẻ html hay không )


        Chú ý :
        trong các thuộc tính CSS luôn có giá trị mặc định
        ví dụ thuộc tính background-color ( không màu )
        thuôc tính background-image ( none ) không có ảnh nền
        thuộc tính background-repeat ( repeat ) luôn luôn lặp lại
        Khi giá trị mặc định có viết hay không cũng không có tác dụng


        Cú pháp background-image :
        background-image: url("đường dẫn của hình ảnh");
