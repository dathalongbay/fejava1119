Cú pháp của CSS
        - Selector ( bộ chọn CSS ) : được sử dụng để chọn 1 hoặc nhiều thẻ
        html
        - Thuộc tính CSS : Thuộc tính để thiết lập màu sắc , màu nền , hình nền ,
        cách hiển thị ...
        - Giá trị của thuộc tính :
        ví dụ thuộc tính màu sắc : xanh , đỏ , tím , vàng ...
        thuộc tính font-size : 12px, 18px ...
        Cú pháp của CSS
        Selector {
            tên thuộc tính 1 : giá trị của thuộc tính 1;
            tên thuộc tính 2 : giá trị của thuộc tính 2;
            tên thuộc tính 3 : giá trị của thuộc tính 3;
            ...
        }

        Selector đầu tiên :
        * ( chọn tất cả các thẻ html )
